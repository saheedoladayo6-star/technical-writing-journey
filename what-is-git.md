# What is Git?
## Introduction
Git is a version control system used to track changes in files. It helps developers and technical writers manage projects efficiently.
## Why Use Git?
Git allows you to:
- Track file changes 
- Work with other people 
- Restore previous versions
- Keep projects organized 
## Common Git Commands
| Command | Purpose|
|---------|--------|
| git innit | Creates a new repository |
| git status | Shows the repository status |
| git add . | Stages changes |
| git commit | Saves changes |
| git push | Upload changes to GitHub |
## Conclusion 
Git is one of the most important tools for developers and technical writers.
