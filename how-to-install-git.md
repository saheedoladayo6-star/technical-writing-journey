# How to Install Git on Windows
## Introduction 
Git is a version control system that helps you track changes to your files. This guide explains how to install Git on a windows computer.
## Prerequisites
Before you begin make sure you have:
- A Windows computer
- An internet connection 
- Administrator access
## Step 1: Download Git
1. Open your web browser.
2. Visit https://git-scm.com.
3. Click **Download for Windows**.
## Step 2: Install Git 
1. Open the downloaded installer.
2. Click **Next** through the set up.
3. CLICK **Install**.
4. Click **Finish** after the installation is complete.
## Step 3: Verify the installaion 
Open command Prompt or VS Code Terminal and run: ```git --version```
If Git is installed correctly, you'll see the installed version number.
## Conclusion
You have succesfully installed Git on your Windows computer.
