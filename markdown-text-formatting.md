# Markdown Text Formatting
## Introduction
Markdown provides several ways to emphasize text and make documents easier to read.
## Bold
### Syntax 
```markdown
**Bold text**
```
### Example
**Bold text**
## Italic
### Syntax
```markdown
*Italic text*
```
### Example
*Italic text*
## Bold and Italic
### Syntax
```markdown
***Bold and italic***
```
### Example
***Bold and Italic***
## Strikethrough
### Syntax
```markdown
~~Deleted text~~
```
### Example
~~Deleted text~~
## Inline Code
### Syntax
```markdown
'git status'
```
### Example
Use `git status` to check your repository.
## Tips
- Use **bold** for important words.
- Use *italic* for emphasis.
- Use `inline code` for commands and filenames.
- Don't overuse formatting.
