# Markdown Blockquotes and Horizontal Rules

## Introduction

Blockquotes highlight important information, while horizontal rules separate sections of a document.

---

## Blockquotes

### Syntax

```markdown
> This is a blockquote.
```

### Example

> **Important:** Always save your work before pushing to GitHub.

---

## Horizontal Rules

### Syntax

```markdown
---
```

### Example

First section.

---

Second section.

## Tips

- Use blockquotes for notes, warnings, and important information.
- Use horizontal rules to separate major sections.
- Don't overuse either feature.
