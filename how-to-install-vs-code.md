# How to Install Visual Studio Code on Windows

## Introduction
**Visual Studio Code is Microsoft's free, fast, and powerful code editor. It supports extensions, debugging, and Git integration, making it an excellent tool for developers and tecnical writers**
## Requirements 
- Windows 10 or 11 {64bit preferred}
- 1.6 GHz + 1GB RAM {4GB+ recommended}
- Internet connection for download
## Steps
1. Go to the official site: https://code.visualstudio.com/
2. Click **Download for Windows** {User setup recommended}.
3. Run the downloaded installer .
4. **Accept the license,choose location{default is fine } and Select all Additional Task "especially Add to PATH".**
5. Click **Install** and **Finish** to launch.
## Verify the Installation  
- Open VS Code from the start menu.
- **Open command Prompt and run:**
code --version
**or open a folder in command Prompt and run:**
code .
## Conclusion 
Installation is complete. VS Code is ready for use, install extensions as needed for your workflow.



## Tips
- Keep VS Code updated.
- Install useful extensions.
- Save your work regularly.
