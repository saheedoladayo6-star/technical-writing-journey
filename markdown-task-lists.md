# Markdown Task Lists

## Introduction

Task lists help track progress by showing completed and incomplete tasks.

## Syntax

```markdown
- [ ] Incomplete task
- [x] Completed task
```

## Example

- [ ] Write the introduction
- [ ] Add screenshots
- [x] Review the document
- [x] Publish to GitHub

## Tips

- Use task lists to organize work.
- Mark completed tasks with `x`.
- Keep tasks short and specific.
